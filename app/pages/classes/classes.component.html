Class

Atributul class specifica unul sau mai multe nume ale unor clase pentru elementele HTML. Numele clasei poate fi folosit de CSS si JavaScript pentru a efectua anumite sarcini asupra elementelor care fac parte din acea clasa. Cand folosim CSS, pentru a selecta elementele dintr-o clasa cu un anumit nume, vom scrie in stylesheet (sau intre tagurile style din interiorul head-ului) caracterul "." urmat de numele pe care l-am acordat clasei:

TOATE tagurile care au numele unei anumite clase vor avea aceleasi proprietati. Pentru a diferentia doar o culoare, sau un tip de text, se va crea o clasa noua, cu un nume nou.

Pentru a combina cele doua clase, atributului class i se vor atribui ambele nume ale claselor, despartite doar prin cate un spatiu, astfel el le va combina proprietatile pe cat posibil.

Nu uitati ca atributul class poate fi folosit si in JavaScript, pentru diferite functii. Apasati pentru a vedea despre ce este vorba.



Id

Atributul "id" specifica un id UNIC pentru un element HTML (valoarea trebuie sa fie unica in cadrul documentului HTML!). Ce inseamna valoare unica? Inseamna ca in cadrul unui document, putem avea mai multe elemente care au un id, dar numele atribuit id-urilor respective trebuie sa fie unic.

Cand folosim CSS, pentru a selecta elementele cu un id specific, vom scrie in stylesheet sau in style caracterul "#", urmat de numele id-ului:

Un element HTML poate avea un singur id,unic,care apartine acelui singur element,in timp ce numele unei clase poate fi folosit de elemente multiple.

Semne de carte folosind id-urile

Semnele de carte HTML,numite "bookmarks" se folosesc pentru a le permite cititorilor sa sara la o parte specifica a paginii web.Acestea pot fi foarte folositoare,mai ales cand pagina este foarte lunga,de exemplu in cazul nostru. Pentru a crea un astfel de semn, prima data trebuie sa creezi bookmark-ul, apoi sa ii adaugi un link. Cand link-ul este apasat, pagina va cobori la locatia semnului de carte .Pentru a face acest lucru aveti de urmat cativa pasi simpli:

  1. Creati semnul cu ajutorul atributului id:

    < h2 id="C4" >Capitolul 4< / h2 >

  2. Adaugati un link semnului de carte ("Mergi la capitolul 4"), de pe aceeasi pagina ,sau de pe o alta pagina:

    < a href="#C4"> Mergi la capitolul 4 < / a > sau < a href="*numele paginii*.html#C4"> Mergi la capitolul 4 < / a >
