Ce este html?


HTML este un limbaj de creare a paginilor web. Aceasta prescurtare vine de la Hyper Text Markup Language.

Elementele html sunt reprezentate prin taguri, de exemplu < p >, < title >, etc., aceste taguri nefiind afisate pe pagina,ci se afiseaza doar continutul lor.

Tagurile sunt de doua feluri: de deschidere si de inchidere, astfel daca deschideti un tag, de exemplu < p > ,acesta trebuie si inchis cu < / p >.

Exista si taguri libere, acestea fiind cele care nu au un continut, deci nu trebuie inchise, de exemplu < br >, care defineste o linie de pauza sau < hr > care delimiteaza imagini, texte etc. printr-o linie orizontala.

  1. Declararea < !DOCTYPE html > defineste documentul ca fiind de tip HTML5 si ajuta browserele sa afiseze pagina corect -HTML5 reprezinta ultima versiune html ( 2004 )

  2. Elementul < html > este elementul de baza (radacina) unei pagini HTML
  3. Tagul < head > contine diferite informatii despre pagina
  4. Tagul < title > reprezinta titlul paginii
  5. Tagul < body > reprezinta continutul vizibil al paginii
  6. Tagul < h1 > defineste cea mai mare rubrica
  7. Tagul < p > defineste un paragraf

Cum creezi o pagina web?

In primul rand trebuie sa folosesti un editor de texte, apoi sa urmezi acesti pasi simpli:

  1. Deschide editorul de texte pe care vrei sa-l folosesti
  2. Foloseste tagurile invatate mai sus si exemplul din imagine pentru a scrie in html
  3. Salveaza pagina ca "index.html", verifica ca tipul fisierului sa fie "all types" sau "html,htm,.." si seteaza codarea de tip UTF-8
  4. Vizualizeaza pagina dand dublu click pe fisier, apoi alege browserul cu care doresti sa-l deschizi
