  • In interiorul divului,cu ajutorul atributului style,veti selecta lungimea si latimea dorita pentru harta:

  • Vom crea o functie JavaScript, pentru a dispune harta. Aceasta functie va fi adaugata cu ajutorul tagului < script >
  • mapCanvas Widgetul Canvas pentru hărți este probabil cel mai important widget din cadrul QGIS(aplicație pentru sistemele informaționale geografice de tip desktop open-source care acceptă vizualizarea, editarea și analiza datelor geospatiale.), deoarece afiseaza harta compusă din mai multe straturi de harti suprapuse și permite interacțiunea cu acestea.
  • mapOptions defineste proprietatile hartii
  • center specifica unde sa centreze harta(folosind coordonatele de latitudine si longitudine, care va vom arata in cele ce urmeaza cum sa le obtineti).
  • zoom specifica nivelul zoom-ului pentru harta, adica cat de aproape sa se vada locatia
  • google.maps.Map este harta in sine, creata cu mapCanvas si mapOptions ca si parametrii

  • Adaugam API-ul de la Google. Un API (Application Programming Interface) este un set de unelte care pot fi folosite pentru construirea aplicatiilor software. Functionalitatea hartii provinde de la o librarie JavaScript localizata la Google. API-ul Google maps ne permite sa afisam harti pe pagina noastra web. Pentru a face acest lucru vom adauga urmatoarele:
    < script src=" https://maps.googleapis.com/maps/api/js?key=YOUR_KEY&callback=myMap " >< / script>
