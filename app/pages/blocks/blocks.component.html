Fiecare element HTML are o valoare implicita de afisare, care depinde de ce fel de element este.

Pentru majoritatea elementelor, aceasta este de tip BLOCK sau INLINE.

Block

Elementele cu o valoare de tip block vor incepe mereu pe o linie noua si vor ocupa intreaga latime posibila (se vor intinde de la stanga la dreapta pe cat de mult se poate). Elementul < div > este un astfel de element, de tip block. De asemenea, pana si tagurile < h1 >-< h6 >,< p >,< ol>,< ul > sau < table > sunt astfel de elemente de tip block.

Elementul < div > se foloseste de obicei ca un "recipient" al altor elemente HTML. Acesta nu are atribute, dar se pot adauga diferite proprietati, cu ajutorul tagului style, sau chiar clase ori id-uri. Inline

Un element de tip inline nu incepe pe o linie noua, desi textul este scris pe linii diferite. Inauntrul unui paragraf, elementul < span > este de tip inline. Probabil ati observat anterior multe din cuvintele cheie scoase in evidenta printr-o culoare diferita. Acest lucru a fost realizat cu ajutorul span-ului, caruia i-am atribuit culoarea alba. Span-ul va colora cu alb tot ce se afla intre el si desi cuvintele vor fi scrise pe randuri diferite, el le va afisa pe unul singur.
